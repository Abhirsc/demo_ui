# Getting Started

Welcome to **Bharat Biodiversity Intelligence**.

This article is loaded from a markdown file so you can edit it directly in VS Code without touching the React code.

## What this platform is for

BBI is being shaped as a workspace for:

- biodiversity data ingestion
- species and occurrence exploration
- climate and spatial layer preparation
- biodiversity insights and attribution-ready outputs

## Basic flow

1. Open the **Data** tab.
2. Import biodiversity records from **GBIF** or **iNaturalist**, or search prepared climate raster layers.
3. Review imported data in **Table View** or **Map View**.
4. Move prepared datasets into **Analysis** and later into **Insight**.
5. Use **Attribution** to support FAIR-aware citation and provenance.
